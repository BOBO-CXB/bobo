## 三、项目启动

### 前端

- yarn 下载所有依赖包
- config/proxy.ts 修改后端服务器地址(仅本地开发需要修改，线上通过nginx跳转)
- npm run dev 运行开发环境
- npm run build打包，打完包在dist目录下

### 后端

- 后台运行需要依赖redis,postgresql,elasticsearch,redis和pg可以使用最新版本,es的版本不能高于8.0,否则项目里面es的客户端jar包无法正常使用
- 后端主工程的docker目录下有使用docker compose进行一键启动部署脚本，将docker-compose.yml和start-dev-env.sh 上传至服务器，运行start-dev-env.sh
- 配置文件位于hliot-standalone/src/main/resources/application-dev.yml 配置文件里面的redis,pg,es地址端口
- 启动org.hliot.pro.standalone.HlIotApplication main方法
- 打包 运行script/build.sh脚本使用maven自动打包（如果报错把 ./mvnw 改为 mvn）,会在根目录下生成hliot-installer.tar.gz，里面的start.sh和stop.sh是启动和停止脚本。

[Service]
Type=forking
PIDFile=/usr/local/nginx/logs/nginx.pid
ExecStartPre=/usr/local/nginx/sbin/nginx -t -c /usr/local/nginx/conf/nginx.conf
ExecStart=/usr/local/nginx/sbin/nginx -c /usr/local/nginx/conf/nginx.conf
ExecReload=/usr/local/nginx/sbin/nginx -s reload
ExecStop=/usr/local/nginx/sbin/nginx -s stop
ExecQuit=/usr/local/nginx/sbin/nginx -s quit
PrivateTmp=true
[Install]
WantedBy=multi-user.target

#启动nginx服务

#### 物联网平台nginx配置文件

```shell
server {
    listen 8080;
    # gzip config
    gzip on;
    gzip_min_length 1k;
    gzip_comp_level 9;
    gzip_types text/plain text/css text/javascript application/json application/javascript application/x-javascript application/xml;
    gzip_vary on;
    gzip_disable "MSIE [1-6]\.";
    location / {
        root /opt/hliot-front;
        index  index.html;
    }

    location ^~/api/ {
        if ($request_uri ~* ^/api/(.*)$) {
           # proxy_pass http://192.168.0.67:8844/$1;
           proxy_pass http://127.0.0.1:8844/$1;
        }
        #proxy_pass http://host.docker.internal:8840/;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP  $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_connect_timeout   1;
        proxy_buffering off;
        chunked_transfer_encoding off;
        proxy_cache off;
        proxy_send_timeout      30m;
        proxy_read_timeout      30m;
        client_max_body_size    500m;
    }
}
```
