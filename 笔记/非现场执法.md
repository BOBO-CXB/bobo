# 非现场执法

- 执法流程（flowStep）：0--违法线索、1--技术审查、2--立案调查、3--法制审核、4--执法文书下达、5--电子归档

- ### 违法线索

  - 由企业监管-异常预警-线索判别生成
  - 线索维护：可查看企业排放数据（异常数据前后12个小时内的排放数据）、用电数据、企业排污许可证、工况数据等，可对线索进行异常调查和提交技术审核等操作
    - 异常调查：确认该条记录不成立为一条线索，退回企业监管-异常调查模块
    - 提交技术审核
      - 上传报告视频等其他证据，上传后的文件信息存放在**this.fileTableData(List)**中
      - 更新线索状态**updateFlowStep**   flowStep = 1
      - 更新附件路径  **updateIllegalFileUrl** filter.fileUrl = JSON.stringify(urlobj)

- ### 技术审查

  - 审核线索
    - 是否通过 **if_approve**  0 未通过1 完全通过 2 基本通过   **通过后才更新flowStep = 2--立案调查**
    - 是否现场执法**if_xczf** 是否现场执法 0 否 1 是
    - 审查阶段 **stage** 审查阶段 0 一级审核 1 二级审核 2 三级审核
    - 在**技术审查表(t_review)**中新增一条审查数据 如果已经存在审查数据则进行更新

- ### 立案调查 

  - 查询的数据为**技术审查表(t_review)** **if_approve = 1 if_xczf =  0 ** 的数据  即**技术审查通过且不需要现场执法的数据**
  - 通知  
    - 更新线索数据的  **if_notice = 1**
  - 处置  
    - 在**t_investigation**新增一条调查数据
    - 如果证据闭环  即**ifClosedCycle=1** **更新flowStep = 3--法制审核**
